# Network-Socket-Communication

This is a project I developed for my Computer Networks course. It’s a client-server application designed to handle real-time data exchange. My goal was to get hands-on experience with socket programming and understand how to manage connections between a server and multiple clients.

## Project Structure:
- **Server:** Written in Python; handles incoming connections and manages data flow.
- **Client (GUI):** A basic interface built in Python to allow user interaction.
- **Client (CLI):** A simple Node.js script for quick connectivity testing in the terminal.
- **Data:** A `questions.json` file used for testing the structured data transmission[cite: 2].

The code is straightforward and demonstrates the fundamental logic of socket-based communication.
